1/**
2 * @file file_input_mode.c
3 * @brief Handles RTCM MSM4 file input, step-by-step parsing and position resolution.
4 *
5 * This function is triggered when the user selects "File Input" from the application menu.
6 * It opens a pre-recorded RTCM file (text or binary), reads each epoch's MSM4 message(s),
7 * extracts relevant GNSS observations, and calculates the receiver position using
8 * the least squares method.
9 *
10 * The resulting receiver position is output in geodetic coordinates (decimal degrees).
11 *
12 * All major processing stages (file handling, message reading, observation parsing,
13 * position solving, and coordinate conversion) are handled by separate modules
14 * for clarity and maintainability.
15 */
