Detailed Description

Parses RTCM 1019 and 1074 messages from text-formatted input lines.

This module provides parsing logic for the following specific RTCM message types:

  • RTCM 1019: Broadcast ephemeris messages (used for GPS satellite position)
  • RTCM 1002: Legacy observation messages (used for GPS L1 pseudorange and phase)
  • RTCM 1074: MSM4 observation messages (used for GPS L1 pseudorange and phase)

The input is assumed to be text-format (e.g., exported from parsed binary logs), and each line contains a full RTCM message with labeled fields (DFxxx).

Output is stored in well-defined structures for further positioning analysis.

double compute_pseudorange_msm1 ( double amb,
double rem )

Computes the pseudorange for an RTCM 1002 (MSM1) observation.

Calculates the pseudorange by combining the rough range integer (in milliseconds) with the pseudorange remainder (in meters), applying the appropriate speed-of-light scaling to the integer term.

Formula: pseudorange = (amb * 299792.458) + rem

Parameters
ambRough range integer in milliseconds.
remPseudorange remainder in meters.
Returns
Pseudorange in meters.
