Utility macro to extract a field from a line by DF label.

This macro simplifies repeated code where a specific DFxxx field value needs to be parsed from a text-formatted RTCM message. It uses strstr() to locate the field, and sscanf() to extract its value.

Parameters
fieldThe DF field label as a string (e.g., "DF009")
formatThe sscanf-compatible format string (e.g., "%lf")
targetA pointer to the variable where the value should be stored
