Detailed Description

Computes the full satellite orbit for each satellite using the EPH data.

For each PRN that has ephemeris, we sweep true anomaly f from 0..2π, compute PQW coordinates (in meters), then rotate to ECI with: Rz(-ω) -> Rx(-i) -> Rz(-Ω)
