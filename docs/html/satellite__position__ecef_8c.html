Detailed Description

Converts the satellite position from ECI coordinates to ECEF frame.

Mirrors the Python implementation: theta = (t / 86400) % 1 * 2*pi (solar day, not sidereal) and performs ecef = eci * Rz(theta). Since our helper does column-vector math (out = R * v), we multiply by Rz^T to match the Python row*matrix result.
